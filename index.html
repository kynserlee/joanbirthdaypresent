Happy Birthday to Joan!

12-19-2016 is the 24th birthday of Joan Lim Wan Ying. Lee Soon Yik, the mighty legend of super heroic expert in programming in Cheras Perdana, want to make something for her. But the normal birthday gifts is too mainstream in Lee's Opinion. Therefore, Lee try to do something different to shows that he got the heart, and hope Joans happy or touched when she see the birthday presents.

What is this?

Basically is a game developed after some sleepless nights. The game is about the things happenned to Joan when she want to eat something with her friend. The game is considered a RPG genre game, but the game is a bit cacat and several bugs because of the limitation of the programmer. But in general it is playable in any platform (windows, mac os and andriod phone).

How to play?

First of all, download the file which from the link given you can see on top(unless you are blind). And then,put it on desktop and extract the zip file. The zip file have password, the passowrd is combination of our birthday date (for example 1219XX11XXXX). After extract the file, simply run the Game.exe then you can play. I don't know how noob Joan in gaming, so i just set to only 5 keys, arrow key to move, click "Z" to proceed dialogue or choose option.
